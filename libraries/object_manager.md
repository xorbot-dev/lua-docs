## get\_valid\_objects

Returns a list of all valid objects.

```lua
object_manager.get_valid_objects( ): table<object>
```

```lua
-- example usage

objs = object_manager.get_valid_objects()
for i,v in ipairs(objs) do print(tostring(v:get_index())) end
```
